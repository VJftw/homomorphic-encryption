This application presents various Homomorphic Encryption schemes to help teach how they work and could be implemented.

More information

Homomorphic Encryption has become an increasingly important topic in Computer Science with the emergence of public cloud services like Amazon Web Services, Google Cloud Platform and Microsoft Azure. It describes the ability to encrypt values and allow any external process to perform operations on them whilst retaining their original meaning.
